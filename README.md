# Meridian

AI-powered productivity and stress management platform built for Gen Z and Millennials.

## Features

* Personalized onboarding
* Gen Z & Millennial specific UI/UX
* Mood and burnout tracking
* Focus & productivity systems
* Daily goals and streaks
* AI coaching companion (“Sage”)
* Progress analytics with charts
* Responsive modern glassmorphism UI

## Tech Stack

* React
* JSX
* Recharts
* CSS Animations
